# Examples

- **`explore.py`** — a guided, interactive tour of the dataset (a [marimo](https://marimo.io)
  notebook). Pick a match and clip and every view re-renders: a synchronised 4-player video grid with
  a keyboard overlay, a top-down arena radar, a HUD, and the per-frame game state.

  ```bash
  pixi run explore     # read-only app view
  pixi run edit        # editable (headless on :2718; SSH-tunnel to view)
  ```
